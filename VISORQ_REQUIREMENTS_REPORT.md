# Visorq 企业级项目组合管理系统 - 需求与规划报告

**版本**: 1.0  
**日期**: 2025-12-11  
**状态**: 生产就绪 (Phase 9 完成)

---

## 1. Executive Summary (执行摘要)

Visorq 是一款现代化的企业级项目组合管理系统 (PPM)，旨在通过直观的可视化界面、智能的资源调度算法和深度的数据分析能力，解决企业在多项目并行管理中面临的资源冲突、成本失控和决策盲区问题。

当前版本 (v1.0.0) 已完成核心业务闭环，涵盖项目全生命周期管理、精细化资源规划、全维度成本控制及 AI 辅助决策系统。系统采用 React 18 + TypeScript + Vite 技术栈，具备优秀的性能和扩展性。

## 2. 已实现功能 (Implemented Features)

### 2.1 项目管理模块 (Project Management)

**核心目标**: 实现项目从立项到交付的全流程数字化管理。

*   **项目全景沙盘 (Dashboard)**
    *   **功能**: 展示项目总数、整体进度、健康度分布、预算执行率等核心 KPI。
    *   **特性**: 支持骨架屏加载，自适应深色模式，实时数据计算。
*   **项目列表与生命周期**
    *   **功能**: 项目增删改查 (CRUD)，支持列表视图和看板视图 (Kanban)。
    *   **特性**: 虚拟滚动 (Virtualized List) 支持大量数据流畅展示；多因子优先级评分模型 (P0-P3)。
*   **项目详情深度视图**
    *   **健康度仪表盘**: 综合时间、成本、质量、范围、风险和团队情绪六维评分。
    *   **交互式甘特图 (Interactive/Professional Gantt)**: 支持任务依赖连线、拖拽调整工期、关键路径显示、里程碑标记。
    *   **任务管理**: 提供看板视图、网络图 (Network Diagram) 和画布视图 (Canvas)，支持复杂依赖关系编辑。
*   **模板与标准化**
    *   **模板管理**: 支持创建标准项目模板，一键复用任务结构和资源需求。
    *   **批量导入**: 支持 CSV/Excel 批量导入项目数据，自动校验格式。

### 2.2 资源管理模块 (Resource Management)

**核心目标**: 最大化资源利用率，实时预警和解决资源冲突。

*   **统一资源池 (Unified Resource Pool)**
    *   **功能**: 管理全员技能标签、职级、部门及可用性。
    *   **特性**: 支持资源画像，展示技能雷达图和历史绩效。
*   **智能分配与冲突检测**
    *   **实时冲突预警**: 在分配时自动检测资源超载 (Overload)，并在时间轴上高亮显示冲突时段。
    *   **技能匹配分析**: 基于项目需求自动推荐最佳人选，计算匹配度得分 (Skill Matching Score)。
*   **可视化资源视图**
    *   **资源热力图 (Heatmap)**: 直观展示团队成员在不同时间维度的负载饱和度。
    *   **增强型时间轴**: 细粒度查看资源在多项目间的具体排期。
    *   **交付效率分析**: 监控资源产出比，分析交付瓶颈。

### 2.3 成本控制模块 (Cost Control)

**核心目标**: 精确核算项目成本，监控预算执行情况。

*   **成本核算体系**
    *   支持人力成本 (基于工时费率)、设备成本、外包成本等多维度核算。
    *   支持预算 (Budget) vs 实际成本 (Actual) 的实时对比。
*   **高级趋势分析**
    *   **EVM (挣值管理)**: 自动计算 PV, EV, AC, CPI, SPI 等专业指标。
    *   **成本预测**: 基于当前趋势预测完工估算 (EAC) 和完工尚需估算 (ETC)。

### 2.4 决策支持与分析 (Decision Support)

**核心目标**: 利用数据驱动高层管理决策。

*   **AI 辅助决策**
    *   **风险评估**: 自动扫描进度延期、成本超支和资源风险，生成风险矩阵。
    *   **智能建议**: 提供资源调优建议和进度追赶策略。
*   **高级报表系统**
    *   支持生成多维度交叉报表。
    *   支持数据导出为 PDF/CSV 格式，满足汇报需求。

### 2.5 用户体验与系统基础

*   **性能优化**: 路由懒加载、代码分割、骨架屏加载策略、虚拟列表。
*   **个性化**: 完整支持深色模式 (Dark Mode)，国际化 (i18n 中英文)。
*   **数据安全**: 基于 LocalStorage 的本地持久化与数据备份/恢复机制。
*   **交互**: 全局通知中心、键盘快捷键支持。

## 3. 未来规划需求 (Future Roadmap)

*   **移动端适配增强**:

*   **全局搜索系统**:
    *   [ ] 实现跨模块搜索 (项目、任务、资源、文档)。
    *   [ ] 支持模糊匹配和高级筛选语法。
*   **第三方集成**:
    *   [ ] **IM 集成**: 消息推送到 Slack/钉钉/企业微信。
    *   [ ] **日历同步**: 支持与 Google Calendar / Outlook 同步日程。
    *   [ ] **Jira/GitLab 同步**: 支持导入外部开发任务状态。
*   **实时协作**:
    *   [ ] 引入 WebSocket，实现多用户同时编辑同一项目时的实时状态同步。

### 3.3 Phase 12: 智能化与自动化 (长期)

*   **AI 预测引擎 2.0**:
    *   [ ] 基于历史项目数据训练模型，更准确地预测工期和风险。
    *   [ ] 实现自然语言查询 (NLP)，例如："显示下个月空闲的高级前端工程师"。
*   **自动化工作流**:
    *   [ ] 允许用户配置触发器 (Trigger) 和动作 (Action)，如"当进度延期超过 10% 时自动发送邮件给 PMO"。
