# Visorq 项目功能点全景脑图文档

> **版本**: 1.0.0
> **生成日期**: 2025-12-16
> **描述**: 本文档深入剖析 Visorq 系统的所有源码与文件结构，拆解出最细粒度的用户功能需求点，并以思维导图的层级结构呈现。

## 1. 核心项目管理 (Project Management Core)

### 1.1 PMO 仪表盘 (Dashboard)
- **KPI 概览**
  - [ ] 查看项目总数统计
  - [ ] 查看不同状态（进行中、规划中、已完成、暂停）的项目数量分布
  - [ ] 查看整体项目健康度分布图表（绿/黄/红状态）
  - [ ] 监控总体预算执行率（预算 vs 实际花费）
- **趋势分析**
  - [ ] 查看项目交付速率趋势图
  - [ ] 查看资源利用率历史趋势
- **快捷入口**
  - [ ] 快速创建新项目
  - [ ] 快速查看我的待办任务
  - [ ] 查看最近访问的项目列表

### 1.2 项目列表与操作 (Project List)
- **列表视图**
  - [ ] 分页/虚拟滚动展示所有项目
  - [ ] 显示项目关键信息：名称、负责人、起止时间、状态、优先级、进度条
  - [ ] 计算并显示项目综合评分（基于多因子模型）
- **看板视图 (Kanban)**
  - [ ] 按项目状态（UI列）拖拽流转项目
  - [ ] 卡片式展示项目摘要信息
- **筛选与搜索**
  - [ ] 按项目状态筛选
  - [ ] 按优先级（P0-P3）筛选
  - [ ] 模糊搜索项目名称或描述
- **项目操作**
  - [ ] **新建项目**：
    - [ ] 填写基本信息（名称、描述、起止日期）
    - [ ] 设置初始优先级与状态
    - [ ] 配置战略对齐度权重
  - [ ] **批量操作**：
    - [ ] 批量删除选定项目
    - [ ] 批量导出项目数据
  - [ ] **单体操作**：
    - [ ] 编辑项目详情
    - [ ] 删除单个项目

### 1.3 项目详情深度视图 (Project Detail)
- **项目概览**
  - [ ] **六维健康度雷达图**：展示进度、成本、质量、范围、风险、团队情绪评分
  - [ ] **基线对比**：查看当前进度与基线（Baseline）的偏差
  - [ ] **阶段门径 (Stage-Gate)**：查看项目当前所处阶段及关口审批状态
- **任务管理 (Task Management)**
  - [ ] **任务列表**：增删改查任务节点
  - [ ] **任务详情**：设置前置依赖、负责人、工期、进度
  - [ ] **WBS 结构**：支持多层级任务分解
- **交互式甘特图 (Interactive Gantt)**
  - [ ] **可视化排程**：时间轴展示任务条
  - [ ] **拖拽操作**：拖动任务条调整时间，拖动端点调整工期
  - [ ] **依赖连线**：可视化创建“结束-开始”等依赖关系
  - [ ] **关键路径**：高亮显示影响项目交付的关键路径（Critical Path）
  - [ ] **里程碑**：特殊的菱形标记显示里程碑节点
  - [ ] **自动排程**：基于依赖关系自动重算后续任务时间
- **画布视图 (Canvas)**
  - [ ] 自由拖拽任务节点进行所见即所得的流程编排
  - [ ] 连线定义任务逻辑关系

### 1.4 标准化管理 (Standardization)
- **模板管理 (Template Manager)**
  - [ ] **查看模板**：浏览预置模板（Web应用、移动端、基础设施等）
  - [ ] **自定义模板**：将现有项目保存为新模板
  - [ ] **应用模板**：基于模板快速初始化项目（复制任务结构、资源需求、风险清单）
- **批量导入 (Batch Import)**
  - [ ] 下载标准 Excel/CSV 导入模版
  - [ ] 上传文件进行数据解析
  - [ ] 预览解析后的数据
  - [ ] 执行批量创建，并反馈成功/失败数量

## 2. 资源智能管理 (Intelligent Resource Management)

### 2.1 资源池 (Resource Pool)
- **资源列表**
  - [ ] 查看所有资源/团队成员
  - [ ] 显示资源的关键属性：职位、技能标签、所属部门、时薪
- **资源画像**
  - [ ] **技能雷达图**：可视化展示资源的能力维度
  - [ ] **历史绩效**：查看该资源在过往项目中的评分
  - [ ] **可用性日历**：查看资源的假期和已分配时段

### 2.2 资源规划与分配 (Allocation & Planning)
- **需求定义**
  - [ ] 在项目中定义所需角色、技能要求、数量及持续时间
- **智能推荐**
  - [ ] 系统根据技能匹配度（Skill Match Score）自动推荐最佳人选
  - [ ] 系统根据可用性（Availability）过滤候选人
- **分配操作**
  - [ ] 将具体人员分配给项目任务或资源需求
  - [ ] 支持“软预定”（Soft Booking）和“硬预定”（Hard Booking）

### 2.3 冲突检测与解决 (Conflict Management)
- **实时预警**
  - [ ] 自动检测同一资源在同一时间段被过度分配（Overload > 100%）
  - [ ] 在时间轴上用红色高亮冲突区域
- **解决建议**
  - [ ] 系统建议替代资源
  - [ ] 系统建议调整任务时间以避开繁忙期

### 2.4 可视化分析 (Resource Visualization)
- **资源热力图 (Heatmap)**
  - [ ] 按周/月展示整个团队的负载饱和度
  - [ ] 颜色深浅表示负载高低
- **统一资源时间轴**
  - [ ] 跨项目查看所有资源的排期甘特图
  - [ ] 拖拽调整跨项目分配

## 3. 风险与质量管理 (Risk & Quality)

### 3.1 风险登记册 (Risk Register)
- **风险识别**
  - [ ] 录入风险：标题、描述、类别（技术/进度/成本等）
  - [ ] 风险评估：设置发生概率（Probability 1-5）和影响程度（Impact 1-5）
  - [ ] 系统自动计算风险等级（Risk Score = P * I）
- **应对策略**
  - [ ] 指定责任人（Owner）
  - [ ] 制定缓解措施（Mitigation Plan）和应急计划（Contingency Plan）
  - [ ] 追踪风险状态（已识别、监控中、已缓解、已关闭）

### 3.2 交付效率与质量 (Delivery Efficiency)
- **交付指标**
  - [ ] 监控需求交付周期（Cycle Time）
  - [ ] 监控缺陷密度（Defect Density）
  - [ ] 监控返工率（Rework Rate）
- **质量仪表盘**
  - [ ] 查看代码评审通过率趋势
  - [ ] 查看自动化测试覆盖率趋势

## 4. 成本与高级分析 (Cost & Analysis)

### 4.1 成本核算 (Cost Control)
- **多维度估算**
  - [ ] 录入人力成本、设备、材料、管理费等多种成本类型
- **预算监控**
  - [ ] 设定项目总预算 (BAC)
  - [ ] 实时对比实际成本 (AC) 与预算
  - [ ] 成本超支预警

### 4.2 挣值管理 (EVM Analysis)
- **指标计算**
  - [ ] 自动计算 PV（计划价值）、EV（挣值）、AC（实际成本）
  - [ ] 自动计算偏差：SV（进度偏差）、CV（成本偏差）
  - [ ] 自动计算绩效指数：SPI（进度绩效）、CPI（成本绩效）
- **未来预测**
  - [ ] 计算 EAC（完工估算）：基于当前绩效预测最终成本
  - [ ] 计算 ETC（完工尚需估算）：预测还需多少投入

### 4.3 组合分析 (Portfolio Analysis)
- **战略对齐分析**
  - [ ] 气泡图展示项目组合的战略价值 vs 风险 vs 成本
- **依赖关系网 (Network Graph)**
  - [ ] 可视化展示跨项目的依赖链路
  - [ ] 识别环形依赖和风险传递路径

### 4.4 高级报表 (Advanced Reporting)
- **报表生成**
  - [ ] 预置报表：项目状态周报、资源利用率月报、财务决算表
  - [ ] 自定义筛选报表时间范围和项目范围
- **数据导出**
  - [ ] 支持导出为 PDF 用于打印汇报
  - [ ] 支持导出为 CSV/Excel 用于二次分析

## 5. AI 与智能决策 (AI Intelligence)

### 5.1 AI 决策看板 (AI Decision Dashboard)
- **风险预测**
  - [ ] AI 扫描历史数据，预测当前项目可能的延期概率
  - [ ] AI 识别潜在的预算溢出风险
- **资源优化建议**
  - [ ] AI 建议资源调配方案以平衡负载
  - [ ] AI 识别技能缺口并建议培训或招聘

### 5.2 智能助手
- **自动生成**
  - [ ] 基于项目描述自动生成风险清单模板
  - [ ] 基于项目类型自动推荐任务分解结构 (WBS)

## 6. 系统基础与体验 (System Foundation)

### 6.1 用户与权限 (Auth & Roles)
- **登录鉴权**
  - [ ] 用户名/密码登录（预置 Admin/Manager/User/Readonly 角色）
  - [ ] 会话持久化
- **权限控制 (RBAC)**
  - [ ] Admin：全功能访问
  - [ ] Manager：项目管理与编辑
  - [ ] User：任务查看与状态更新
  - [ ] Readonly：仅查看权限

### 6.2 个人工作台 (User Workbench)
- **个人中心**
  - [ ] 查看“分配给我的任务”
  - [ ] 查看“我负责的风险”
  - [ ] 快捷更新任务进度和状态

### 6.3 系统设置 (Settings)
- **偏好设置**
  - [ ] 切换深色/浅色主题 (Dark Mode)
  - [ ] 切换系统语言 (中/英) - i18n
- **通知系统**
  - [ ] 查看系统内通知消息（Toast + 通知中心列表）
  - [ ] 管理已读/未读状态

### 6.4 数据安全 (Data Security)
- **本地持久化**
  - [ ] 所有业务数据自动保存到 LocalStorage
  - [ ] 页面刷新不丢失数据
- **数据管理**
  - [ ] **数据备份**：下载完整系统数据快照 (JSON)
  - [ ] **数据恢复**：上传快照恢复系统状态
