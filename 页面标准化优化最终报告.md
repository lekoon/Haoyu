## 🎓 最佳实践

### 1. 始终使用PageContainer
```tsx
// ✅ 正确
<PageContainer>
  {/* 内容 */}
</PageContainer>

// ❌ 错误
<div className="min-h-screen bg-slate-50">
  {/* 内容 */}
</div>
```

### 2. 使用PageHeader统一头部
```tsx
// ✅ 正确
<PageHeader
  title="标题"
  description="描述"
  actions={<Button>操作</Button>}
/>

// ❌ 错误
<div className="flex justify-between">
  <h1>标题</h1>
  <button>操作</button>
</div>
```

### 3. 优先使用StatCard展示KPI
```tsx
// ✅ 正确
<StatCard title="总数" value={100} icon={Icon} />

// ❌ 错误
<div className="bg-white p-6">
  <p>总数</p>
  <h3>100</h3>
</div>
```
