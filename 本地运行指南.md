# Visorq 本地运行指南

本文档介绍如何在本地环境获取、安装并运行 Visorq 项目。

## 1. 环境准备

在开始之前，请确保您的电脑上安装了 **Node.js**。
- 推荐版本：Node.js 18.0 或更高版本
- 检查方法：在终端输入 `node -v`

## 2. 获取代码

如果您还没有克隆仓库，请执行以下命令：

```bash
git clone https://github.com/lekoon/Visorq.git
cd Visorq
```

## 3. 安装依赖

在项目根目录下，运行以下命令安装所需的第三方库：

```bash
npm install
```
*注意：这可能需要几分钟时间，取决于您的网络状况。*

## 4. 启动本地服务器

安装完成后，运行以下命令启动开发服务器：

```bash
npm run dev
```

启动成功后，您会看到类似以下的输出：

```
  VITE v5.x.x  ready in 350 ms

  ➜  Local:   http://localhost:5173/Visorq/
  ➜  Network: use --host to expose
```

👉 **打开浏览器访问**：按住 `Ctrl` 点击链接，或在浏览器地址栏输入 `http://localhost:5173/Visorq/` 即可使用。

## 5. 其他常用命令

- **构建生产版本**：
  ```bash
  npm run build
  ```
  这会在 `dist` 目录下生成用于部署的静态文件。

- **本地预览生产构建**：
  ```bash
  npm run preview
  ```
  这可以模拟生产环境运行构建后的代码。

## 6. 常见问题

**Q: 启动时提示端口被占用？**
A: Vite 会自动尝试下一个可用端口（如 5174），请留意终端输出的实际地址。

**Q: 页面显示空白？**
A: 请确保您访问的 URL 包含 `/Visorq/` 后缀（因为项目配置了 base path）。正确的地址通常是 `http://localhost:5173/Visorq/`。
