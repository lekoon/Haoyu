# 资源交付效率看板使用指南

## 概述
资源交付效率看板是一个集成了资源管理与项目交付效率分析的综合可视化平台。它旨在帮助管理者从"资源管理"转向"效率优化"，通过多维度数据分析识别瓶颈并提供智能决策支持。

## 核心功能模块

### 1. 全局效率视图 (Overview)
采用"6指标法则"设计的核心KPI仪表盘，以及三维散点矩阵图。
- **三维散点矩阵图**:
  - **X轴**: 事务吞吐量 (Throughput) - 反映交付速度
  - **Y轴**: 事务前置时间 (Lead Time) - 反映交付周期
  - **Z轴 (气泡大小)**: 事务颗粒度 - 反映任务规模
  - **颜色**: 资源利用率 (红=过载, 绿=合理, 蓝=正常)
- **使用场景**: 快速识别"低吞吐量、高前置时间、高资源占用"的问题项目。

### 2. 资源-效率关联分析矩阵 (Matrix)
交互式的交叉分析工具，揭示资源投入与产出效率的关系。
- **矩阵结构**: 行(项目) x 列(资源类型)
- **可视化编码**:
  - **单元格颜色**: 资源利用率 (三色预警)
  - **单元格高度**: 与交付效率成反比 (效率越低，格子越大，越显眼)
- **交互功能**: 支持点击展开，查看具体的资源瓶颈分析和关联影响。

### 3. 实时监控预警系统 (Alerts)
基于动态阈值的智能监控中心。
- **多级预警**: 黄色(关注) -> 橙色(警告) -> 红色(紧急)
- **影响分析**: 自动计算资源异常对项目前置时间的具体影响 (例如: "预计导致延期 15%")
- **动态阈值**: 系统根据历史数据自动调整合理区间 (DWAFE模型概念实现)。

### 4. 智能优化建议 (Optimization)
AI驱动的决策辅助模块。
- **优化建议**: 提供资源调整、流程优化和优先级调整的具体建议。
- **方案模拟**: 预览优化方案对资源利用率和交付效率的预期提升效果。
- **一键执行**: 支持快速采纳建议 (模拟功能)。

## 如何访问
1. 进入 **资源管理中心** (Enhanced Resources Dashboard)。
2. 点击右上角的 **"交付效率看板"** 切换按钮。
3. 使用顶部的视图选择器在不同模块间切换。

## 技术实现
- **前端框架**: React + TypeScript
- **可视化库**: Recharts (高性能图表渲染)
- **图标库**: Lucide React
- **数据处理**: date-fns (时间序列处理)
- **架构设计**: 模块化组件设计，支持独立扩展和维护。
