# CTPM Tool - 管理层快速参考指南

## 🎯 核心功能速览

### 1. 高级仪表板（Executive Dashboard）

**访问路径**: 登录后默认页面

**关键指标**:
- 📊 **项目组合健康度**: 0-100 分综合评分
- ⚠️ **风险项目**: 30 天内到期的活跃项目
- 🎯 **高优先级项目**: 评分 > 8.0 的项目
- 👥 **资源利用率**: 实时显示百分比

**快速操作**:
```
1. 点击"Export Report"导出 CSV 报告
2. 查看彩色柱状图了解项目优先级
3. 查看饼图了解项目状态分布
4. 查看资源分配表识别瓶颈
```

---

### 2. 数据备份与恢复

**访问路径**: Settings → Data Management

**备份操作**:
```
1. 点击"Backup Now"按钮
2. 保存下载的 JSON 文件到安全位置
3. 建议文件命名：ctpm-backup-YYYY-MM-DD.json
```

**恢复操作**:
```
1. 点击"Restore"按钮
2. 选择之前的备份文件
3. 确认后系统自动刷新
```

**⚠️ 重要提示**:
- 建议每周备份一次
- 重要操作前先备份
- 备份文件妥善保管

---

### 3. 项目优先级管理

**评分因子管理**:
```
Settings → Scoring Factors
- 调整各因子权重（总和应为 100%）
- 添加新的评分因子
- 删除不需要的因子
```

**项目评分**:
```
Projects → 新建/编辑项目
- 为每个因子打分（0-10）
- 系统自动计算优先级分数
- 自动排名
```

## 📈 关键决策支持

### 资源分配决策

**查看资源状态**:
1. Dashboard → 资源分配分析表
2. 查看利用率颜色：
   - 🟢 绿色（<70%）：健康
   - 🟡 黄色（70-90%）：高负荷
   - 🔴 红色（>90%）：超载

**行动建议**:
- 超载资源：考虑招聘或外包
- 低利用率：重新分配或培训转岗

### 项目优先级决策

**Top 5 项目**:
- Dashboard 底部显示最高优先级项目
- 🥇 金色：第一名
- 🥈 银色：第二名
- 🥉 铜色：第三名

**风险项目处理**:
1. 查看"At Risk"卡片
2. 识别 30 天内到期项目
3. 采取行动：加速、延期或取消

## 🎨 界面说明

### 颜色编码系统

**项目状态**:
- 🔵 蓝色：Planning（规划中）
- 🟢 绿色：Active（进行中）
- 🟠 橙色：On Hold（暂停）
- 🟣 紫色：Completed（已完成）

**优先级分数**:
- 🟢 绿色（>8.0）：高优先级
- 🔵 蓝色（5.0-8.0）：中优先级
- ⚪ 灰色（<5.0）：低优先级

**健康度评分**:
- 🟢 绿色（>70）：健康
- 🟡 黄色（50-70）：需关注
- 🔴 红色（<50）：需改进

## 🔧 常见问题

### Q: 如何提高项目组合健康度？
A: 
1. 提高项目平均分数（优化评分）
2. 提高项目完成率（按时交付）
3. 优化资源利用率（70-85% 最佳）

### Q: 数据存储在哪里？
A: 
- 浏览器 localStorage（本地）
- 不会自动同步到云端
- 必须定期备份

### Q: 如何添加新项目？
A:
1. Projects → New Project
2. 填写基本信息
3. 分配资源
4. 为各因子评分
5. 保存

### Q: 如何修改评分标准？
A:
1. Settings → Scoring Factors
2. 调整权重滑块
3. 确保总和为 100%
4. 点击"Recalculate All Scores"
