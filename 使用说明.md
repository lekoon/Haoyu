# CTPM Tool - 使用说明

## 快速开始

### 访问网站
在线地址：**[https://lekoon.github.io/CTPMtool/](https://lekoon.github.io/CTPMtool/)**

### 登录
默认管理员账号：
- **用户名**: `admin`
- **密码**: `admin123`

- 修改个人信息

- 查看账户信息

## 操作指南

### 如何创建新项目
1. 点击左侧菜单的"项目"
2. 点击"新建项目"按钮
3. 填写项目信息（名称、描述、开始日期等）
4. 点击"保存"

### 如何添加资源
1. 点击左侧菜单的"资源"
2. 点击"添加资源"按钮
3. 填写资源信息（姓名、技能、可用性等）
4. 点击"保存"

### 如何添加新用户
1. 使用管理员账号登录
2. 点击左侧菜单的"设置"
3. 在"用户管理"部分点击"添加用户"
4. 填写用户信息（用户名、密码、角色等）
5. 点击"保存"

### 如何修改密码
1. 点击右上角的用户头像
2. 选择"个人资料"
3. 在"修改密码"部分输入当前密码和新密码
4. 点击"保存"

### 如何退出登录
1. 点击右上角的用户头像
2. 选择"退出登录"

## 数据说明

### 数据存储
- 所有数据存储在浏览器的本地存储（localStorage）中
- 数据不会自动同步到其他设备或浏览器
- 清除浏览器数据会导致所有记录丢失

### 数据备份
建议定期备份重要数据：
1. 打开浏览器开发者工具（F12）
2. 进入"Application"或"应用程序"标签
3. 在左侧选择"Local Storage"
4. 复制并保存数据

## 常见问题

### Q: 忘记密码怎么办？
A: 目前系统不支持密码重置功能。如果忘记密码，需要清除浏览器的 localStorage 数据，系统会重置为默认管理员账号。

### Q: 为什么我的数据丢失了？
A: 可能的原因：
- 清除了浏览器缓存和数据
- 使用了不同的浏览器或设备
- 浏览器隐私模式（无痕模式）

### Q: 可以在手机上使用吗？
A: 可以，但建议在电脑上使用以获得最佳体验。

### Q: 支持多人协作吗？
A: 当前版本不支持多人实时协作。每个用户的数据存储在各自的浏览器中。

### Q: URL 中的 # 符号是什么？
A: 这是正常的。系统使用 Hash 路由来确保在 GitHub Pages 上正常运行。

## 技术支持

如遇到问题，请检查：
1. 浏览器版本是否为最新
2. 是否启用了 JavaScript
3. 是否清除了浏览器缓存
4. 网络连接是否正常
